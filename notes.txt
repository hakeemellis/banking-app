Random Notes to Help Me Remember

To install Angular:

npm install -g @angular/cli
ng new "app-name"
cd "app-name"
ng serve

ng serve: to run app


To update Angular:

ng update @angular/cli @angular/core


To access firebase cli in angular:

ng add @angular/fire


For Further Understanding:

src/app/components: Contains Angular components related to the application's UI.
src/app/services: Contains services responsible for handling data and business logic.
src/app/models: Contains TypeScript models representing data structures used in the application.
src/assets: Contains static assets like images, fonts, and data files.
src/environments: Contains environment configuration files for development and production.
src/index.html: The main HTML file for the application.
src/main.ts: The entry point of the application.
src/styles.scss: Global styles for the application.
angular.json: Angular CLI configuration file.
package.json: Contains project dependencies and scripts.
tsconfig.json: TypeScript configuration file.

<router-outlet></router-outlet> : loads nested routes in Angular
